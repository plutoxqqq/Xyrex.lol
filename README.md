# Xyrex.lol

**Xyrex.lol** is an executor hub that displays details about different executors—including pricing, official websites, features, and other relevant information. It helps users compare options and discover trustworthy details about their favorite executors.

> https://xyrex.lol

---

## 🚀 Overview

The platform organizes executor data in a searchable and filterable format, including:

- Executor names
- Pricing tiers (free, paid, or both)
- Basic details (stability, trust, etc.)
- Feature summaries
- Filtering tools for easier discovery
- Mini-games

Xyrex helps users compare tools without jumping between multiple websites.
